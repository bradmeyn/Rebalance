# Rebalance

## Summary

Rebalance is a portfolio management application that allows users to track their investments and ensure their portfolio stays inline with target holding allocations. It also provides watchlist functionality allowing users to add investments to a personal watchlist along with a target price.

## Features

- Kotlin based Android application conforming to Material Design guidelines
- User registration & authentication using Firebase Auth
- Interaction with Yahoo Finance API to get live prices utilising Retrofit & Moshi libraries
- CRUD functionality for portfolio and watchlist
- Modern MVVM architecture utilising viewmodels w/livedata, repositories & entity dao's via Room ORM for database interactions
- Swipe gestures for deleting and editing
- Charts using MPAndroid Charts
